4. 吸烟者问题

三个吸烟者在一个房间内，还有一个香烟供应者。为了制造并抽掉香烟，每个吸烟者需要三样东西：烟草、纸和火柴，供应者有丰富货物提供。三个吸烟者中，第一个有自己的烟草，第二个有自己的纸，第三个有自己的火柴。供应者随机地将两样不同的东西放在桌子上，允许一个吸烟者进行对健康不利的吸烟。当吸烟者完成吸烟后唤醒供应者，供应者再随机地把两样不同的东西放在桌子上，唤醒一个吸烟者。试用信号量的P、V操作设计该问题的同步算法，给出所用共享变量（如果需要）和信号量及其初始值，并说明各自的含义。

 15.00 得分：15.00

int num=0;//吸烟者编号

semaphore offer1=0;

semaphore offer2=0;

semaphore offer3=0;

semaphore finish=0;

int random();

void P(semaphore);

void V(semaphore);



void Provider(){

    while (1) {

        num=random()%3;

        if (num==0) {

            V(offer1);//提供烟草和纸

        }else if(num==1){

            V(offer2);//提供烟草和火柴

        }else if(num==2){

            V(offer3);//提供纸和火柴

        }

        //将两样不同的东西放在桌子上

        P(finish);

    }

}

    }

}

        P(offer3);

        //拿纸和火柴

        V(finish);

    }

}

定义了四个信号量offer1、offer2、offer3、finish初始值置为0，共有四个进程分别为一个生产者和三个吸烟者。生产者随机生成一个编号并为该编号的吸烟者提供材料，所以执行对应的V(offeri)操作，当将准备好的东西放在桌子上后执行P(finish)等待某吸烟者吸完烟。对于某吸烟者进程来说，首先消耗生产者提供的材料，即执行P(offeri)操作，然后执行V(finish)操作，从而让生产者可以继续生产。
